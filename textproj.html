Project Overview

For this project I used SpaCy to analyze a block of text from a Sherlock Holmes book. I explored using pre-processing techniques to collect meaningful data about the text. For example, finding the top 15 most frequently meaningful words used.

import operator

            def print_top_words(word_freq_dict, num_words=15):
                sorted_word_freq = dict(sorted(word_freq_dict.items(), key=operator.itemgetter(1), reverse=True))
                top_words = list(sorted_word_freq.items())[:num_words]
                print(f"Top {num_words} most frequent meaningful words:")
                for word, count in top_words:
                    print(f"{word}: {count}")

Final Result
